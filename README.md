## Requirement
* Docker (version: 20.10.7)
* Docker Compose (version: 1.29.2)

## Installation
```bash
curl -s https://bootstrap.pypa.io/get-pip.py | python
curl -s https://get.docker.com/ | sh
service docker start
pip install docker-compose
```
## Run environment
```bash
docker-compose up
```
